How It Works

SwanFlow uses an ESP32-CAM microcontroller with an OV2640 camera module to capture traffic images. The SIM7000A LTE module transmits data over 4G to our cloud backend.

Powered entirely by a 20W solar panel and 12V battery, each monitoring station operates autonomously without external power connections.

Edge ML models trained with Edge Impulse perform vehicle detection directly on the ESP32, minimizing data transmission and enabling real-time counting.
