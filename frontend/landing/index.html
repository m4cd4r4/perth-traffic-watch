Hardware

Edge Device

  • Camera: OV2640 2MP sensor
  • Resolution: QVGA 320√ó240 (FOMO optimized)
  • Capture Rate: 1 fps detection interval
  • Power: 5V 2A (~500mW typical)
  • Connectivity: SIM7000A 4G LTE

Enclosure

  • Housing: IP65 junction box
  • Mounting: Pole/wall bracket
  • Weather Rating: IP65 (weatherproof)

Machine Learning

Model Architecture

  • Framework: Edge Impulse FOMO
  • Backend: TensorFlow Lite Micro (int8)
  • Input: 96√ó96 grayscale
  • Inference: 100-200 ms on ESP32
  • Footprint: <100 KB RAM

Training

  • Dataset: 300-500 labeled images
  • Classes: Vehicle, Background
  • Accuracy: >70% real-world
  • F1 Score: 0.65-0.85
